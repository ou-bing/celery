=================================
 Celery - 分布式任务队列
=================================

Celery 是一个简单、灵活且可靠的分布式系统，用来处理海量消息，同时还提供了工具箱用来维护维护这种系统。

它是一个专注于实时处理的任务队列，同时还支持任务调度。

Celery 有一个庞大且多样化的用户和贡献者社区，你可以加入我们的 :ref:`IRC <irc-channel>`
或我们的 :ref:`邮件列表 <mailing-list>`。

Celery 是开源的，使用 `BSD License`_。
